| File | What | Use |
|---|---|---|
| `logo-light.svg` / `logo-light.png` | Dark glyph for light backgrounds | README in light mode, light website surfaces |
| `logo-dark.svg` / `logo-dark.png` | Light glyph for dark backgrounds | README in dark mode (via `<picture media="(prefers-color-scheme: dark)">`) |
| `logo-512.png` | 512×512 square logo | Connector listing icon (Anthropic / OpenAI directories), favicon source |

## Usage

- Don't modify the logo glyphs. Color tweaks for backgrounds are fine; glyph edits aren't.
- Don't combine the logo with another brand's logo without asking. The consent screen pattern (Claude + ShinobiData) is the canonical co-branding.
- For social cards / blog posts, link back to <https://shinobidata.com>.
